# Описание репозитория
Небольшой первый проект в фреймворке Qt осуществляющий прогноз погоды на текущий день в конкретном месте. Основной код был написан на языке С++ используя библиотеки Qt для создания связи программы и сервера ([openweathermap](https://openweathermap.org)). 
Также для большего удобства были написаны классы WeatherUnit и WeatherRequest. Большое спасибо за классы Request и RequestSender пользователю [@ArmanPrestige](https://habr.com/ru/users/ArmanPrestige/) с Хабра!
